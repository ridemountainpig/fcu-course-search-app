# FCU Course Search APP

A APP for [FCU Course Search](https://github.com/ridemountainpig/fcu-course-search).

## Technologies

- EXPO
- React Native
- NativeWind

## Setup Project

```bash
# install expo cli
npm install -g expo-cli

# install dependencies
yarn install

# run in expo
npm start
```
